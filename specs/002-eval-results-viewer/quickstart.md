# Quickstart: Eval Results Viewer

## Prerequisites
- Node.js (v18+)
- Local `tests` directory containing evaluation data

## Setup

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run the Viewer**
   The viewer expects a `tests` directory in the project root containing `eval_dataset`, `eval_results`, and `llm_invoke_results`.
   ```bash
   npm run view
   ```
   *This command will start the local Express API server and the React frontend, then automatically open `http://localhost:3000` in your browser.*

## Architecture Overview
- **Backend**: A lightweight Node.js/Express server that reads and parses local JSON/JSONL files on-demand.
- **Frontend**: A React application that fetches data from the local API and renders the summary list and detailed chat UI.
- **Data Source**: The application is entirely file-backed, relying on the outputs generated by the evaluation framework.
