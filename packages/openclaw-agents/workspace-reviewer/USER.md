# User: Reviewer

## Review Preferences
- Review Frequency: Weekly (周度复盘) + Ad-hoc (单笔复盘)
- Focus Areas: Risk Management (风控合规), Entry Timing (入场时机), Pattern Recognition (模式识别)
- Review Depth: Full analysis with data attribution (完整分析含数据引用)

## Grading Standards
- A (9-10): Excellent — correct direction, proper risk control, good timing
- B (7-8): Good — mostly correct, minor issues
- C (5-6): Average — some mistakes but overall acceptable
- D (3-4): Poor — significant errors in judgment or risk management
- F (0-2): Fail — major violations, ignored signals, or catastrophic loss

## Report Format
- Language: 中文为主，技术术语保留英文
- Structure: Summary → Decision Analysis → Risk Compliance → Timing → Grade → Lessons → Actions
- Must include: Comparative tables (plan vs actual), specific data citations, pattern detection
